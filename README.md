# Machine Learning Fundamentals

A collection of machine learning and statistical learning projects implemented as part of hands-on coursework and self-study.

## Topics Covered

- k-Nearest Neighbors (KNN) Classification
- Gaussian Modeling and Classification
- Principal Component Analysis (PCA)
- Kalman Filtering and Object Tracking

## Repository Structure

```text
.
├── 01_knn-classification.ipynb
├── 02_gaussian-classification-iris.ipynb
├── 03_pca-indian-pines.ipynb
└── 04_kalman-filter-tracking.ipynb
```

## Project Overview

### 1. KNN Classification
Implementation and evaluation of the k-Nearest Neighbors algorithm for supervised classification tasks.

### 2. Gaussian Classification
Probabilistic classification using multivariate Gaussian models, applied to the Iris dataset.

### 3. PCA for Hyperspectral Data Analysis
Dimensionality reduction and visualization of hyperspectral imagery using Principal Component Analysis (PCA).

### 4. Kalman Filter Tracking
State estimation and trajectory tracking of moving objects using Kalman filtering techniques.
